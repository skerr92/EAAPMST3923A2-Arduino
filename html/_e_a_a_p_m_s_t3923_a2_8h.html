Detailed Description

This is a library for the EAAPMST3923A2 Ambient Light & Proximity Sensor

Designed specifically to work with the Oak Development Technologies EAASTPM3923A2 based products -—> urlhere

The sensor uses I2C to communicate with additional pins that can be connected to for other operational modes.

Writen by Seth Kerr for Oak Development Technologies
